系統介紹

商品瀏覽

顯示清單與詳情，支援關鍵字 / 品牌等條件擴充。

購物車管理

加入 / 調整 / 移除商品，動態計算小計與總金額，銜接結帳流程。

訂單查詢

建立訂單、查詢歷史訂單與明細，模擬小型電商平台使用情境。

使用技術

Frontend
  • HTML5 / CSS3
  • Bootstrap（RWD）
  • jQuery + AJAX（呼叫後端 API）
Backend & DB
  • Spring Boot（REST API）
  • Spring Data JPA（資料存取）
  • MySQL（商品/訂單/會員資料）
  • Maven、GitHub、Postman
